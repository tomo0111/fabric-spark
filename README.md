# Getting Started
 * Environment Apache Spark on CentOS7

# Commands
```
$ fab -l
Available commands:

    java.install_java
    scala.install_scala
    scala.setup_scala
    spark.install_spark
    spark.setup_spark
    wget.install_wget
```

# Single node environment
Install Java
```
$ fab java.install_java
```

Install Scala
```
$ fab scala.install_scala
```

Install wget
```
$ fab wget.install_wget
```

Setup scala
```
$ fab scala.setup_scala
```

Install Spark
```
$ fab spark.install_spark
```

Setup Spark
```
$ fab spark.setup_spark
```
